SpringBoot 集成Dubbo

概述

我们常常使用 Springboot 暴露 HTTP 服务，并走 JSON 模式。但慢慢量大了，一种 SOA 的治理方案。这样可以暴露出 Dubbo 服务接口，提供给 Dubbo 消费者进行 RPC 调用。
优点

  • Dubbo 不单单只是高性能的 RPC 调用框架，更是 SOA 服务治理的一种方案。
  • 远程通信，向本地调用一样调用远程方法。
  • 集群容错
  • 服务自动发现和注册，可平滑添加或者删除服务提供者。
